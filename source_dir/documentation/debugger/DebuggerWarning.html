DebuggerWarning()

Syntax

DebuggerWarning(Message$)
Description
Generates a runtime debugger warning. Can be useful when creating re-usable modules meant to be shared.

Parameters

Message$ The warning message to display.

Return value

None.

See Also

DebuggerError()

Supported OS

All
