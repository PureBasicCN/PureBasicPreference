PureBasic真的小巧并且速度很快？
是. All has been optimized to give maximum speed and compactness to the programs created with PureBasic. The execution speed match almost any professional compiler like Visual C++ and the executable size is smaller, without the need for any runtime (small programmes are typically between 5 kb and 10 kb).
PureBasic有可视化编辑器？
是. 一个强大的 Visual Designer包含在安装包中并且允许你创建任何种类的应用程序. It supports all PureBasic gadgets, includes a visual menu bar editor, status bar support, multi-language facilities, mutli-window, realtime properties updates and more...
当我购买了PureBasic，我可以免费获得所有更新？?
是. The very first PureBasic user (which has registered in 1998) still has free access to new updates and this is not going to change (it only applies if registered on this web site, not for boxed version of PureBasic).
PureBasic的许可在所有操作系统都有效？
是. The PureBasic licence is an user-based licence, which means you can install your full version on all of your computers without the need to buy another licence for each. PureBasic currently runs on four different operating systems (Windows, Linux, MacOS X and AmigaOS) and a single licence cover them all.
我使用PureBasic创建商业应用？
是. 一旦你拥有PureBasic，你可以做任何一种程序（免费软件，共享软件，甚至商业应用），而不需要支付任何额外费用到 Fantaisie Software。
难道是允许使用在其他项目的PureBasic制成的DLL？
一般来说是的。You can make DLLs including PureBasic commands for your own projects without any restrictions. But it's not allowed to release simple "wrapper" Dlls to include PureBasic commands in other programming languages.
Are all the versions always up-to-date ?
Except AmigaOS which is deprecated, all other versions are released in sync, with the same features.
为什么 Windows API 说明不包含在 PureBasic 文档中 ?
Even if PureBasic supports natively nearly all of the Windows API functions, it's impossible to include all their descriptions as it's more than 1 GB of data. You can get all the needed informations on the Microsoft site.
Are there different PureBasic packages available and where can I buy them ?
否. PureBasic is currently only available in a download version directly on this site. Earlier, there was boxed versions, which were not freely updatetable. If you are interested you can upgrade to the full version for a special price. Look in the order section for more details.
How fast do i get the package delivered after ordering ?
After you've paid the registration fee you will get your package in about 20 minutes as all is processed automatically. Be sure to check your spam folder as well, sometimes the automatic mail does end in it !
