PureBasic 是基于既定的BASIC规则本机 32 位和 64 位编程语言。PureBasic 的主要特点是可移植性 （Windows、 Linux 和 MacOS X 目前均支持），生成非常快速和高度优化的可执行文件，当然，BASIC也语法非常简单。PureBasic 针对初学者创建并且专业人士也非常喜欢。我们有大量的精力投入，形成快速、 可靠的系统友好语言及其实现。

尽管它的语法很适合初学者，但是对于PureBasic的高级特性是不受限制的，比如指针， 结构，过程，动态链接表和其它.. 有经验的程序员可以访问任何已经定义的OS结构或APi函数，PureBasic甚至允许行内嵌汇编。



PureBasic 主要特点

   - 丰富的内部指令（1400+）可以快速且简单的创建应用和游戏
   - 所有 BASIC关键词都支持
   - 非常快速的编译所创建的可执行文件也是高度优化了的。
   - 创建的可执行文件无需扩展DLL,运行库或其他任何必备因素。
   - 过程支持本地和全局变量的结构编程
   - 完整的 unicode 支持
   - 针对高级程序员支持完整的OSAPI访问
   - 简单但是非常快速的2D游戏支持，通过专用库实现 (DirectX, SDL, ...)
   - 基于OGRE的方便和高质量的3D支持
   - 使用了高度优化的（汇编）指令进行了最佳的可用硬件优优化
   - 源代码在 Windows, MacOS X 和 Linux上通用
   - 专用的编辑器和开发环境
   - 功能强大的集成调试器和分析器可以轻松的跟踪和优化代码
