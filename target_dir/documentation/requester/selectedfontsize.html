SelectedFontSize()

语法

Size = SelectedFontSize()
概要
Returns the size of the font chosen by the user with the FontRequester().

参数

无.

返回值

The size of the font chosen by the user with the FontRequester().

参阅

FontRequester()

已支持操作系统

所有
