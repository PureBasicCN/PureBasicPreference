WriteSerialPortData()

语法

Result = WriteSerialPortData(#SerialPort, *Buffer, Length)
概要
将任意数量的数据写入到指定的串行端口。

参数

#SerialPort 要使用的串口。
*Buffer 要写入的缓冲区。
Length 要写入的缓冲区的长度，以字节为单位。

返回值

返回写入串行端口的字节数，如果此操作失败，则返回 0 。

备注

要检测在输出缓冲区中有多少数据是可用的，请使用函数AvailableSerialPortOutput()。

参阅

OpenSerialPort(), WriteSerialPortString()

已支持操作系统
