SerialPortError()

语法

Result = SerialPortError(#SerialPort)
概要
当使用函数ReadSerialPortData()、 WriteSerialPortData()或WriteSerialPortString()失败时，在串口上返回的错误。

参数

#SerialPort 要使用的串口。

返回值

可以是下列值的组合︰ 
  #PB_SerialPort_RxOver︰ 发生输入的缓冲区溢出。
                         输入缓冲区中没有足够的空间
                         或在文件尾 (EOF)之后接收到字符
  #PB_SerialPort_OverRun︰ 字符缓冲区溢出发生。
                          下一个字符丢失。
  #PB_SerialPort_RxParity ︰ 硬件检测到奇偶校验错误。
  #PB_SerialPort_Frame : 硬件检测到帧错误。
  #PB_SerialPort_Break : 硬件检测到中断条件。
  #PB_SerialPort_TxFull ︰ 应用程序尝试发送一个字符，但输出缓冲区已满。
  #PB_SerialPort_IOE ︰ 与设备在通信期间出现 I/O 错误。
  #PB_SerialPort_WaitingCTS ︰ 指定传输是否正在等待发送 CTS （清除发送） 信号。
  #PB_SerialPort_WaitingDSR ︰ 指定传输是否正在等待发送 DSR （数据准备就绪） 信号。
  #PB_SerialPort_WaitingRLSD ︰ 指定传输是否正在等待 RLSD （接收线信号检测） 发送的信号。
  #PB_SerialPort_XoffReceived︰ 指定传输是否正在等待，因为接收到 XOFF 字符。
  #PB_SerialPort_XoffSent ︰ 指定传输是否正在等待，因为 XOFF 字符被传送。
                               Transmission halts when the XOFF character is transmitted to a system that takes the next
                               character as XON, regardless of the actual character.
  #PB_SerialPort_EOFSent: 指定文件结束（EOF）字符已接收。

参阅

ReadSerialPortData(), WriteSerialPortData(), WriteSerialPortString()
