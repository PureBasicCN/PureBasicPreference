SetSerialPortStatus()

语法

SetSerialPortStatus(#SerialPort, Attribute, Value)
概要
更改指定串行端口状态。

参数

#SerialPort 要使用的串口。
Attribute 要设置的属性。它可以是以下值之一︰ 
  #PB_SerialPort_DTR︰ 设置 DTR 信号状态 （值可以是 0 或 1）
  #PB_SerialPort_RTS︰ 设置 RTS 信号状态 （值可以是 0 或 1）
  #PB_SerialPort_TXD︰ 设置的 TXD 信号状态 （值可以是 0 或 1）
  #PB_SerialPort_XonCharacter ︰ 用于 Xon/Xoff 握手顺序的字符 （值可以是 1 到 255 之间）。默认值为 $11 。
  #PB_SerialPort_XoffCharacter︰ 用于 Xon/Xoff 握手顺序的字符 （值可以是 1 到 255 之间）。默认值为 $13 。
Value 要设置的属性值。根据设定的 '属性' 描述以查看哪个值是适用的。

返回值

无.

示例

  If OpenSerialPort(0, "COM1", 300, #PB_SerialPort_NoParity, 8, 1, #PB_SerialPort_XonXoffHandshake, 1024, 1024)
    SetSerialPortStatus(0, #PB_SerialPort_XonCharacter, 8)
    SetSerialPortStatus(0, #PB_SerialPort_XoffCharacter, 9)
    Debug "Success"
  Else
    Debug "Failed"
  EndIf
