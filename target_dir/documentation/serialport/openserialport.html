OpenSerialPort()

语法

Result = OpenSerialPort(#SerialPort, SerialPortName$, Bauds, Parity, Data, Stop.f, HandshakeMode, InputBufferSize, OutputBufferSize)
概要
打开串口以备使用.

参数

#SerialPort 识别新串口的编码. #PB_Any 可以用来自动生成这个编码.
SerialPortName$ 串口识别符 ，例如在Windows系统是 "COM1" 或者 在Linux系统是 "/dev/ttyS0" .
Bauds 定义串口通信连接速度（波特率），可以是下列值之一︰ 
    50, 75, 110, 150, 300, 600, 1200, 1800, 2400
    4800, 9600, 19200, 38400, 57600 or 115200
Parity 定义通信如何处理奇偶性校验。它可以是下列值之一︰ 
  #PB_SerialPort_NoParity   : No parity  无校验
  #PB_SerialPort_EvenParity : Even parity  偶校验
  #PB_SerialPort_MarkParity : Mark parity  校验位始终为1
  #PB_SerialPort_OddParity  : Odd parity  奇校验
  #PB_SerialPort_SpaceParity: Space parity  校验位始终为0
Data 定义的数据长度，字节 （通常是 7 或 8） 。
Stop 设置停止位 （1、 1.5 或 2）。
HandshakeMode 设置握手的模式。它可以是下列值之一︰ 
  #PB_SerialPort_NoHandshake     : No handshaking
  #PB_SerialPort_RtsHandshake    : No handshaking but RTS is set to 1
  #PB_SerialPort_RtsCtsHandshake : RTS/CTS
  #PB_SerialPort_XonXoffHandshake: Xon/Xoff
InputBufferSize 定义输入缓冲区的大小，以字节为单位。
OutputBufferSize 定义输出缓冲区的大小，以字节为单位。

返回值

如果串行端口已成功打开，返回值为非零，否则零 （它可能已经在使用中，或参数错误）。如果使用#PB_Any为 #SerialPort 参数，返回一个生成的数。

示例

  If OpenSerialPort(0, "COM1", 300, #PB_SerialPort_NoParity, 8, 1, #PB_SerialPort_NoHandshake, 1024, 1024)
    Debug "成功"
  Else
    Debug "失败"
  EndIf
