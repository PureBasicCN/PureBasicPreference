GetSerialPortStatus()

语法

Result = GetSerialPortStatus(#SerialPort, Attribute)
概要
返回指定串口状态.

参数

#SerialPort 要使用的串口。
Attribute 它可以是下列值之一︰ 
  #PB_SerialPort_RI : 获取RI 信号状态 (0 或者 1)
  #PB_SerialPort_DCD: 获取 DCD 信号状态 (0 或者 1)
  #PB_SerialPort_DSR: 获取 DSR 信号状态 (0 或者 1)
  #PB_SerialPort_CTS: 获取 CTS 信号状态 (0 或者 1)
  #PB_SerialPort_XonCharacter : 获取用于  Xon/Xoff  的握手顺序值 ( 1 到 255)
  #PB_SerialPort_XoffCharacter: 获取用于  Xon/Xoff  的握手顺序值 ( 1 到 255)

返回值

根据指定的属性，返回指定的串行端口状态.

已支持操作系统

所有

<- CloseSerialPort() - SerialPort Index - IsSerialPort() ->
