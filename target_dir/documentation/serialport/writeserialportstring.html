WriteSerialPortString()

语法

Result = WriteSerialPortString(#SerialPort, String$ [, Format])
概要
将一个字符串写入指定的串行端口.

参数

#SerialPort 要使用的串口。
String$ 要写入的字符串.
Format (optional) 字符串格式. 它可以是以下值之一︰ 
  #PB_Ascii   : 字符串将以 ascii  的格式写入， (默认, even in unicode mode).
  #PB_UTF8    : 字符串将以 UTF8 格式写入.
  #PB_Unicode : 字符串将以 unicode (UTF16) 格式写入.

返回值

返回写入串行端口的字节数，如果此操作失败，则返回 0 。

备注

要检测在输出缓冲区中有多少数据是可用的，请使用函数AvailableSerialPortOutput()。

参阅

OpenSerialPort(), WriteSerialPortData()

已支持操作系统

所有
