CreateRegularExpression()

语法

结果 = CreateRegularExpression(#RegularExpression, Pattern$ [, Flags])
概要
使用指定的模式创建一个新的正则表达式.

参数

#RegularExpression 一个用来识别这个新生成的正则表达式的数字. #PB_Any 可以用来自动生成这个编码.
Pattern$ 用来识别要进行匹配, 提取或替换的文本的正则表达式.
Flags (可选) 可以是下列值的组合︰ 
  #PB_RegularExpression_DotAll    : '.' 匹配任何包含换行符的文本.
  #PB_RegularExpression_Extended  : 空格和 '#' 注释将会被忽略.
  #PB_RegularExpression_MultiLine : '^' 和 '$' 匹配包含换行的文本.
  #PB_RegularExpression_AnyNewLine: 将 'CR', 'LF', 和 'CRLF' 识别为换行分隔.
  #PB_RegularExpression_NoCase    : 比较和匹配将大小写敏感

返回值

如果正则表达式成功创建则返回非零值,否则返回零. 如果#PB_Any被用来作为 #RegularExpression 参数则成功时返回生成的数字. 如果在当前模式中发生了错误, 则返回值为零. 要了解关于出错的更多内容,请查阅 RegularExpressionError() .

备注

如果一个正则表达式不再被使用, 则 FreeRegularExpression() 命令可以释放一些资源.

示例

  ; 这条表达式将匹配所有以小写字母开头, 
  ; 同时以字母 'b' 连接然后以一个大写字母结束的三字母单词. 例如: abC
  ;    
  If CreateRegularExpression(0, "[a-z]b[A-Z]")
    Debug MatchRegularExpression(0, "abC") ; Will print 1
    Debug MatchRegularExpression(0, "abc") ; Will print 0
  Else
    Debug RegularExpressionError()
