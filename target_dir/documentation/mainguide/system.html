系统必备

PureBasic 可以运行在 Windows XP、 Windows Vista、 Windows 7，Windows 8 和 Windows 10 （ 32 位和 64 位版本）、 Linux (内核 2.2 或以上) 和 MacOS X （10.6 或以上）。

如果有任何问题，请与我们联系。

Windows

对于很多图像函数比如 3D 或 Sprites 则至少需要 DirectX9.0c 版本. Windows Vista 和新版本从开始就没有安装，所以它需要手动安装。

你可以点这里 http://www.microsoft.com/en-us/download/details.aspx?displaylang=en&amp;id=35下载和安装"DirectX 最终用户运行时 Web 安装程序" ，。

Linux

使用PureBasic所需的软件包：

- sdl 1.2 devel (针对游戏)
- gtk 2+ (针对 GUI 程序)
- libstdc++ devel
- gcc 已正确安装

- iodbc 和 iodbc-devel 用于Database 命令 (参见 www.iodbc.org)

- libwebkit 在用到 WebGadget() 时也应该安装.

- xine 和 xine-devel 针对 Movie 命令检查 INSTALL 和 README文件查看更多信息.

MacOS X

您需要安装苹果的开发工具，以便能够使用PureBasic。. 这些工具可以在安装CD或者苹果公司的官网上可以找到： http://developer.apple.com/

应有意识地使用最新版本的工具 (例如 XCode)，根据你的操作系统版本的需要 ！命令行工具需要在XCode安装后安装.
