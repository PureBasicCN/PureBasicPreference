简介

PureBasic 是基于既定的"BASIC"规则"高级别"的编程语言。它兼容于任何其它 "BASIC" 编译器. 学习 PureBasic 非常容易!PureBasic 针对初学者创建并得到了他们和专家的一致好评。编译时间非常快。这个软件已经开发了用于Windows操作系统版本。我们有大量的精力投入研发了一种快速、 可靠和系统友好的语言及其实现。

语法很简单，但具有无限可能的PureBasic的高级函数添加到这个语言中，比如指针，结构，程序，动态链接列表等等. 经验丰富的程序员可以毫不费力的使用OS中的结构或API对象。<br5>

PureBasic 是一种便携式的编程语言，目前在 Linux、 Mac OS X 和 Windows 计算机系统上都可以运行。这意味着，相同的代码可以在所有系统中本地编译并仍然能使用每个系统下的全部功能。这样就没有类似虚拟机或者代码解释器之类的性能瓶颈，生成的代码都针对这个操作系统编译的高度优化后的可执行文件。

PureBasic 的主要特点

- 支持x86 和 x64
- 内置数组，动态链表，复杂结构，指针和变量自定义
- 支持类型有: Byte (8-bit), Word (16-bit), Long (32-bit), Float (32-bit), Quad (64-bit), Double (64-bit) 和字符串
>- 用户自定义类型 (结构)
- 内置字符串类型 (字符数组),包括 ascii 和 unicode
-功能强大的宏支持
-常量，二进制和十六进制数字支持
- 表达式减法器通过分组常量和数字编号在一起
-标准算术支持标志优先和括号: +，-，/，*，and，or ，< <，> >
-极快速编译
   - 过程支持本地和全局变量的结构编程
- 所有标准 <f47><b48>BASIC 关键词: If-Else-EndIf, Repeat-Until, 等等....
- 专业库来操纵 BMP 图片、 windows、 控件，DirectX，等
- 专业的库经过了高度优化为了最快速度和压缩度
-Win32 API 是完全支持，就像他们是基本关键字一样
- 行内嵌汇编
-通过常量文件预编译来实现扩展快速编译
-可配置 CLI 编译器
-非常高的生产力，全面关键词，联机帮助
-系统友好，易于安装和使用
