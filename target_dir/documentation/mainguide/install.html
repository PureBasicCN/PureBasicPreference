安装

要安装 PureBasic, 只需要单击安装 向导,按照步骤操作 然后单击 PureBasic 图标 (可以在 桌面或开始菜单找到) 来启动PureBasic.

要使用 命令 行 编译程序, 开启 一个标准 命令 行 窗口 (CMD)并且在 "Compilers\" 子目录 查找 PBCompiler.exe. 有一个好方法是将 "PureBasic\Compilers\" 目录 添加到 路径 环境 变量 中可以让编译器可以从任何目录访问.

重要提示︰ 为了避免冲突与现有的 PureBasic 安装 （也许还有用户库），请安装一个新的 PureBasic 版本总是在自己新的文件夹中。请参阅使用多个 PureBasic 版本章节。
