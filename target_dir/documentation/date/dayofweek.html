DayOfWeek()

语法

Result = DayOfWeek(Date)
概要
返回指定日期的工作日<星期几>.

参数

Date 需要被提取值的日期.

返回值

返回一个介于 0 到 6 之间的数字，代表一周中的某一天: 
  0 : Sunday【星期日】
  1 : Monday【星期一】
  2 : Tuesday【星期二】
  3 : Wednesday【星期三】
  4 : Thursday【星期四】
  5 : Friday【星期五】
  6 : Saturday【星期六】

示例

  Debug DayOfWeek(Date(2006, 10, 30, 0, 0, 0))  ; 输出 '1' 表示 星期一.

参阅

FormatDate(), Year(), Month(), Day(), Hour(), Minute(), Second()
