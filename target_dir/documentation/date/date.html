Date()

语法

Date = Date([Year, Month, Day, Hour, Minute, Second])
概要
返回给定参数的日期值，如果没有指定参数，则返回当前系统日期.

参数

Year, Month, Day, Hour, Minute, Second (optional) 返回日期的组件. 如果没有指定这些参数，则使用当前的系统日期和时间.

返回值

返回指定的日期值.

示例

  Debug Date() / (3600*24*365)  ; 将输出从1970年01月01日至今的年数
  Debug Date(1999, 12, 31, 23, 59, 59)  ; 将输出 '946684799' ( 从 01/01/1970 0:00:00 到 12/31/1999 23:59:59 的秒数)

参阅

FormatDate(), Year(), Month(), Day(), Hour(), Minute(), Second()
