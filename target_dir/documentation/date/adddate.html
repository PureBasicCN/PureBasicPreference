AddDate()

语法

Date = AddDate(Date, Type, Value)
概要
给日期加一个时间.

参数

Date 需要被加时间的日期值.
Type 添加值的类型. 它可以是下列常量之一: 
  #PB_Date_Year   : Will add 'Value' Years to the date【以<年>为单位添加到日期】
  #PB_Date_Month  : Will add 'Value' Months to the date【以<月>为单位添加到日期】
  #PB_Date_Week   : Will add 'Value' Weeks to the date【以<周>为单位添加到日期】
  #PB_Date_Day    : Will add 'Value' Days to the date【以<天>为单位添加到日期】
  #PB_Date_Hour   : Will add 'Value' Hours to the date【以<小时>为单位添加到日期】
  #PB_Date_Minute : Will add 'Value' Minutes to the date【以<分钟>为单位添加到日期】
  #PB_Date_Second : Will add 'Value' Seconds to the date【以<秒>为单位添加到日期】
注意: 当值类型使用 #PB_Date_Month 时, 程序将自动根据每个月的实际天数来计算添加值, 例如: 2008年03月31日加 1 个月之后结果是2008年04月30日， 因为4月份没有31天.
Value 添加到日期的值. 使用一个负数值可以用来减去日期.

返回值

返回一个被添加值后的新日期.

示例


  Debug FormatDate("%yyyy/%mm/%dd", AddDate(Date(), #PB_Date_Year, 2)) ; Returns the current date + 2 years

  Debug FormatDate("%mm/%dd/%yyyy", AddDate(Date(), #PB_Date_Year, 2)) ; Returns the current date + 2 years
