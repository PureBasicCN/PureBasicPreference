ParseDate()

语法

Date = ParseDate(Mask$, String$)
概要
将字符串日期转换为日期格式，然后其他日期函数可以使用它.

参数

Mask$ 掩码字符串，该字符串定义了日期字符串格式化的方式，用于格式化日期格式如下: 
  %yyyy: 将替换为 年，4位数字.
  %yy: 将替换为 年，2位数字.
  %mm: 将替换为 月，2位数字.
  %dd: 将替换为 日，2位数字.
  %hh: 将替换为 小时，2位数字.
  %ii: 将替换为 分钟，2位数字.
  %ss: 将替换为 秒，2位数字.
String$ 要解析的日期字符串.

返回值

返回表示已解析字符串的日期. 如果输入的字符串与掩码不匹配，则结果为 -1.

示例


  Debug ParseDate("%yy/%mm/%dd", "10/01/07")     ; 返回日期格式 "10/01/07"

  Debug ParseDate("%mm/%dd/%yyyy", "01/07/2010") ; 返回日期格式 "01/07/2010"
