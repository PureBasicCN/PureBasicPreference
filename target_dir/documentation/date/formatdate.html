FormatDate()

语法

Text$ = FormatDate(Mask$, Date)
概要
返回给定日期，以字符串形式表示.

参数

Mask$ 用于格式化日期的掩码. 在掩码字符串中的下面的标记将根据给定的日期替换: 
  %yyyy: 将替换为 年，4位数字.
  %yy: 将替换为 年，2位数字.
  %mm: 将替换为 月，2位数字.
  %dd: 将替换为 日，2位数字.
  %hh: 将替换为 小时，2位数字.
  %ii: 将替换为 分钟，2位数字.
  %ss: 将替换为 秒，2位数字.
Date 需要格式化显示的日期值.

返回值

返回由日期值所替换的所有标记的掩码字符串（根据掩码格式显示日期）.

示例


  Debug FormatDate("Y=%yyyy, M= %mm, D=%dd", Date()) ; 将显示当前系统日期
                                                     ; 输出： "Y=2010, M=01, D=07"

  Debug FormatDate("%mm/%dd/%yyyy", Date()) ; 将显示当前系统日期
                                            ; 输出： "01/07/2010"

  Debug FormatDate("%hh:%ii:%ss", Date())   ; 将以  15:20:02  时间格式显示
