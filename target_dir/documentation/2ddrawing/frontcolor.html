FrontColor()

语法

FrontColor(Color)
概要
Set the default color for graphic functions and text display.

参数

Color The new color to be used as the foreground color. 这个颜色可以是 RGB 或 RGBA 格式. 是否使用alpha通道取决于 绘图模式.

这里有一个通用的颜色参照表.

返回值

无.

参阅

BackColor(), RGB(), RGBA(), DrawingMode()
