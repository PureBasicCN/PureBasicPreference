Point()

语法

Color = Point(x, y)
概要
返回当前输出中指定的某个像素的颜色.

参数

x, y 在输出中像素的位置.

出于性能的原因有没有界限检查这些坐标上执行，指定的坐标必须在当前绘图区域内。OutputWidth() and OutputHeight() can be used to verify that. 通过对ClipOutput()的调用此命令也不受影响。

返回值

返回指定像素的颜色。

如果输出具有 32 位颜色深度和当前的DrawingMode()设置为 alpha 通道模式之一，这种颜色将只包含 alpha 信息. 否则将颜色的 alpha 分量设置为 0.

备注

此命令不能与PrinterOutput()同时使用。
