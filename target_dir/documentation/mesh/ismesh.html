IsMesh()

语法

Result = IsMesh(#Mesh)
概要
Tests if the given #Mesh is valid and the mesh has been correctly initialized.

此函数是保险性的，可能与任何值一起使用。If Result equals zero then the given mesh has not been properly created or initialized. This is the correct way to ensure a mesh is ready to use.

已支持操作系统

所有
