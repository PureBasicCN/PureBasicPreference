GetRuntimeString()

语法

Result$ = GetRuntimeString(Object$)
概要
返回运行时对象的字符串值。

参数

Object$ Name of the object to get the value from. 支持下列对象︰
- Variable:︰ 对象名称是变量名 （不区分大小写）。
- Constant: 对象名称是 '#Constantame' (不区分大小写).

When accessing public module items, the module prefix name is mandatory, even if UseModule is used.

返回值

返回指定的对象或空字符串的字符串值，如果没找到。As an empty string is a valid string value, IsRuntime() can be used to determine if the runtime object really exists.

参阅

SetRuntimeString(), IsRuntime()

已支持操作系统

所有
