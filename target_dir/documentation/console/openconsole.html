OpenConsole()

语法

Result = OpenConsole([Title$])
概要
打开一个控制台窗口。该函数必须在这个库中的任何其他函数之前被调用。只有一个控制台可以在一个的PureBasic程序同时被打开。

参数

Title$ (可选) 新的控制台窗口的标题。在Windows中，指定一个标题允许自定义控制台参数的节能，如字体，颜色等。它在其他操作系统中没有效果。

返回值

如果result 为0，表示已经失败并且必须禁止所有进一步调用到的控制台函数。

备注

控制台关闭可以通过CloseConsole（）函数

同EnableGraphicalConsole（）控制台可以将文本和图形模式之间切换。

在Microsoft Windows中，有两种不同的可执行文件格式：Win32和控制台。If you want to create a standard console application, like 'dir', 'del' etc. you must compile the executable using the 'Console' format (Compiler Option menu in the PureBasic IDE). On Linux and OS X, there is no special Console format however setting the Compiler option to 'Console' will launch a terminal window automatically when you run your program from the IDE.

示例

  OpenConsole()
  PrintN("Waiting 5 secs before quit...")
  Delay(5000)
