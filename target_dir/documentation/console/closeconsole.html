CloseConsole()

语法

CloseConsole()
概要
使用OpenConsole()打开控制台窗口前请先关闭先前的控制台窗口。当年你关闭控制台后，将不能使用任何与控制台有关的函数，除非你再次打开。您的程序结束后控制台窗口会自动关闭。

参数

无.

返回值

无.

示例

  For i = 0 To 4
    If OpenConsole()
      PrintN("这是一个控制台 #"+Str(i))
      PrintN("按回车键关闭控制台")
      Input()
      CloseConsole()
    EndIf
  Next
