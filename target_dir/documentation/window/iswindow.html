IsWindow()

语法

Result = IsWindow(#Window)
概要
Tests if the given #Window number is a valid and correctly initialized, window.

参数

#Window 要使用的窗口.

返回值

返回非零值，如果 #Window 是一个有效的窗口，否则为返回零。

备注

此函数是保险的并且可以与任何值一起使用。如果 'Result'是非零则对象 是 有效的 并初始化了的，否则它将等于零. 这是确保一个窗口是准备好要使用的正确途径。
