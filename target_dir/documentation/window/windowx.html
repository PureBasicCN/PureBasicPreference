WindowX()

语法

Result = WindowX(#Window [, Mode])
概要
返回指定窗口的屏幕上的 x 位置。

参数

#Window 要使用的窗口.
模式 （可选） 用于计算窗口 x 位置的模式。它可以是以下值之一︰ 
  #PB_Window_FrameCoordinate: 窗口的x 位置，包括边框 （默认值） 。
  #PB_Window_InnerCoordinate: x 窗口内区域 （在这里可以添加控件） 的位置
                              不包括边框。

返回值

返回的 x 位置 （从左至右） 在给定窗口的屏幕 （以像素为单位）。

参阅

OpenWindow(), WindowY()
