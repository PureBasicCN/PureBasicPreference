WindowY()

语法

Result = WindowY(#Window [, Mode])
概要
Returns the y position on the screen of the given window.

参数

#Window 要使用的窗口.
模式 （可选） 此模式用于计算窗口的Y位置. 它可以是以下值之一︰ 
  #PB_Window_FrameCoordinate: y 位置的窗口，包括边框 （默认值）。
  #PB_Window_InnerCoordinate: y 位置的窗口内区域 （在这里可以添加小工具），
                              不包括边框。

返回值

返回的 y 位置 （从顶部开始） 从给定窗口的屏幕上 （以像素为单位）。

参阅

OpenWindow(), WindowX()

已支持操作系统
