常用规则

PureBasic 既定的规则，请不要随便改变。分别有:

注释

注释的标志符为“ ;”。在注释符 ; 后所有的文本内容都会在编译程序时被编译器忽略掉。

示例

  If a = 10 ; 这是一个注释说明的内容

关键字

所有 关键字 keywords 在 PureBasic 中都是常用的东西,像创建数组 (Dim) 或列表 lists (NewList), 或是控制程序流 (If : Else : EndIf)。关键字不含有括号 '()', 括号常用于 PureBasic 功能函数 functions使用。

示例

    If a = 1      ; If, Else 和 EndIf 是关键字; 条件 'a = 1'
      ...         ; 是一个表达式中使用的变量。
    Else
      ...
    EndIf
关键字 Keywords 的详细内容可以在帮助手册索引页左侧找到。

函数

所有 函数 functions 后面必须紧跟着括号“( ”，否则将不被视为一个函数。 (函数就是没有参数也要加括号)。

示例

    EventWindow() ; 这是一个函数。
    EventWindow   ; 这是一个变量。
Functions 通常被列入 PureBasic "命令库 Command libraries" 中, 在帮助手册索引页右侧可以找到函数命令列表。

常量

所有的常量前面都有一个标志符号 #。它们只需要在代码中声明一次，就会始终保持其给定的值。(编译器在编译可执行文件时会将所有常量替换成所对应的值。)

示例

  #Hello = 10 ; 这是一个常量。
  Hello  = 10 ; 这是一个变量。

文本字符串

文本字符串声明使用 " 引号符指定。转义序列支持使用 ~ 字符，放在文本字符串的前面。允许的转义序列如下︰ 
  \a: alarm           Chr(7)
  \b: 退格键       Chr(8)
  \f: 换页符        Chr(12)
  \n: 换行符         Chr(10)
  \r: 回车 Chr(14)
  \t: 水平制表符  Chr(9)
  \v: 垂直制表符    Chr(11)
  \": 双引号   Chr(34)
  \\: 反斜杠       Chr(92)

示例

  a$ =  "Hello world"  ; 标准字符串
  b$ = ~"Escape\nMe !" ; 使用转义序列的字符串

标签

所有标签后面必须都紧跟: （冒号）。标签名称不能包含任何运算符 (+,-,...) 以及特殊字符 (ß,ä,ö,ü,...)。当自定义函数 procedure中声明时，此标签仅在此过程中可用。

示例

  I_am_a_label:

表达式

表达式是一个用来计算的求值项，可以混合使用同类型的变更、常量或函数。当在表达式中使用数字时，您可以在它前面添加一个关键字 $ 符号，来表示十六进制数或关键字 % 符号来表示一个二进制数。如果没有这类符号，数将被视为十进制。字符串必须用引号括住。

示例

  a = a + 1 + (12 * 3)

  a = a + WindowHeight(#Window) + b/2 + #MyConstant

  If a <> 12 + 2
    b + 2 >= c + 3
  EndIf

  a$ = b$ + "this is a string value" + c$

  Foo = Foo + $69 / %1001  ;  十六进制数和二进制数的用法

串联命令

任意数量的命令都可以通过使用“:”冒号串联到一行代码里。

示例

  If IsCrazy = 0 : MessageRequester("Info", "Not Crazy") : Else : MessageRequester("Info", "Crazy") : EndIf

行继续符

如果某一代码行包含一个很长的表达式，可将它拆分在几行上。拆分符包含: 加号 (+), 逗号 (,),坚号 (|), And, Or, Xor。

示例

  Text$ = "Very very very very long text" + #LF$ +
          "another long text" + #LF$ +
          " and the end of the long text"

  MessageRequester("Hello this is a very long title",
                   "And a very long message, so we can use the multiline" + #LF$ + Text$,
                   #PB_MessageRequester_Ok)

本手册中的类型词

使用的词语︰
<variable> : 一个Basic 变量。
<expression> : 如上面所述的表达式。
<constant> : 一个数值常量
<label> : 程序标签
<type> : 任何类型, (标准 standard 或 结构 structured).
