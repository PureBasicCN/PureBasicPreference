用户指南 - 第一步 使用Debug调试输出 (变量 & 操作)

我们来个典型的 "Hello World" 吧. 你想看看吗？Ok，看下面两个例子：

Debug 调试输出

在 Debug 窗口输出 "Hello World" : 
  Debug "Hello World!"

对话框输出

在 对话框窗口输出 "Hello World" : 
  MessageRequester("", "Hello World!")

高级Debug调试例子

继续一个简短的例子，使用可用的变量类型，算术运算符并显示结果: 
  a = 5
  b = 7
  c = 3

  d = a + b  ; 将变量  'a' 加 'b' 的和存储到变量 'd'
  d / c ;我们直接使用“d”的值（= 12）除以“c”的值（= 3），并将结果（=4）保存到变量“d”

  Debug d   ; 将输出 4
这种使用变量的方式，“瞬间”你就会了。 要强制编译器总是要求变量第一次使用前必须声明变量，只需使用关键字EnableExplicit。
