用户指南 - 常量

除了变量，PureBasic 也提供了一种方法来定义常量，实际上提供了不止一种方法。我们快速了解一下。PureBasic 本身预定义的常量（所有以#PB_开头），或从 操作系统提供的API 常量。IDE 的Structure Viewer结构查看器工具有一个面板，显示所有预定义的常量。为使代码更具可读性，您可以定义自己的常量，以# 号为前缀。 
  #MyConstant1 = 10
  #MyConstant2 = "Hello, World!"
Enumeration 枚举— — 在一系列常量枚举中，PureBasic 将自动按顺序进行编号，默认情况下，枚举将从零开始 — — 但是如果需要的话也是可以改变的。 
  Enumeration
    #MyConstantA
    #MyConstantB
    #MyConstantC
  EndEnumeration
  
  Enumeration 10 Step 5
    #MyConstantD
    #MyConstantE
    #MyConstantF
  EndEnumeration
  
  Debug #MyConstantA
  Debug #MyConstantB
  Debug #MyConstantC
  
  Debug #MyConstantD   ; 常量值是 10
  Debug #MyConstantE   ; 常量值是 15
  Debug #MyConstantF   ; 常量值是 20
