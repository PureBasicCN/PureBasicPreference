Static


语法
Static[.<type>] <variable[.<type>]> [= <constant expression>] [, ...]
概要
Static 允许创建一个本地持久的allows to create a local persistent variable in a Procedure even if the same variable has been declared as Global in the main program. 如果在= Static之后指定类型， 则默认类型更改为此定义. Static 同样可以用于 数组, 列表 和 映射. 当声明一个静态数组，纬数必须是常量值。

变量的值不在每个过程调用时重新初始化，意味着您可以使用本地变量平行与全局变量 （具有相同的名称），两者都将保持它们的值。每个变量可以有一个默认值，直接分配给它，但它必须是一个常量值。

在Static 旁边你可以使用关键词 Protected, 来分开来自本地变量的全局变量，但是 与 Protected 在一起的本地变量将不会保持他们的值.

示例︰用于变量

  Global a
  a = 10
  
  Procedure Change()
    Static a
    a+1
    Debug "In Procedure: "+Str(a) ; 将打印 1, 2, 3 因为变量在每次过程调用中递增.
  EndProcedure
  
  Change()
  Change()
  Change()
  Debug a ; 将打印 10, 因为静态变量无法影响全局.

示例︰与数组

  Global Dim Array(2)
  Array(0) = 10
  
  Procedure Change()
    Static Dim Array(2)
    Array(0)+1
    Debug "In Procedure: "+Str(Array(0)) ; 将打印 1, 2, 3 因为在每次调用后数组字段会递增.
  EndProcedure
  
  Change()
  Change()
  Change()
  Debug Array(0) ; 将打印 10, 因为静态数组无法影响全局.
