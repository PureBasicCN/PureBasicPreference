用户指南 - 变量和变量处理

A 是一个整型（integer） - 但请注意，如果你用EnableExplicit指令，就不能这样声明变量。在 过程 之外的 A 属于外部变量，过程之内的 A 属于局部变量. 
  A = 0

B 是一个长整型（ long integer）, C 是一个单精度数值型（ floating-point number），它们被初始化为 0 . 
  Define B.l, C.f

D 和 E 也是长整. 然而，它们将被初始化 声明为常量。查看 Define 的其他使用语法. 
  Define.l D = 10, E = 20

F 是一个字符串型（ string）. 
  Define.s F

G$ 如果你使用这种方式声明字符串变量，在后面使用此变量时，就必须始终在变量名后带上$符号. 
  G$ = "Hello, "
以下代码不能正常编译. (G 会被编译器视为一个新的整型变量，并且编译器会抛出一个error错误异常，G$ 和 G 是两个不同的变量). 
  G = "Goodbye, World!"

H 是一个长度为 20 的字符串型 array 数组，数组索从 0 开始（0 到 19）. 
  Dim H.s(19)
这里 H 是一个长度为 25 的字符串型数组. 如果数组大小被调整（ReDim），那它原来存储的值将会被保留. 
  ReDim H.s(24)

J 是一个全局变量（Global）,. 作为一个全局变量，不光会作用于主代码区，同时，也会在后续所有的子函数（procedure）过程中起作用，在主代码区中使用（Shared ）共享关键词，让出现意外的可能性最小化。 
  Global.i J

K 是一个新的，空的 字符串列表 list . 
  NewList K.s()

M 是一个新的，空的 映射（ map）. 
  NewMap M.s()

请注意，您不能使用 Define 关键词来定义 NewList 或 NewMap . 否则，将导致编译器报错.

在过程函数中: 
  Procedure TestVariables()
  �
   ; N and P will be local.
   Define.l N
   Protected.l P
  �
   ; Q will be a static local.
   Static.l Q
  �
   ; The main scope variable F and the string list K will be available within this procedure.
   Shared F, K()
  �
   ; The global scope variable J will be available here implicitly.
  �
  EndProcedure

Using operators on variables: 
  ; Add two to A.
  A + 2
  
  ; Bitwise Or with 21 (A will become 23)
  A | 21
  
  ; Bitwise And with 1 (A will become 1)
  A & 1
  
  ; Arithmetic shift left (A will become 2, that is 10 in binary).
  A << 1

String concatenation: 
  G$ + "World!"

Add an element to the K list: 
  AddElement(K())
  K() = "List element one"

Add an element to the M map: 
  M("one") = "Map element one"
