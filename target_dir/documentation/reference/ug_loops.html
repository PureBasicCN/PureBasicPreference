用户指南 - 循环

数据、 事件或许多其他的事情，可以使用循环来处理，不断检查是否满足执行条件。循环控制使用的关键词: Repeat : Until, Repeat : Forever, While : Wend, For : Next, ForEach : Next.

在此计数器中 每次循环 A 将增加 2 ，这个循环将总是执行相同数量的迭代次数。 
  Define.i A
  For A = 0 To 10 Step 2
     Debug A
  Next A
这个循环，将变量 B 的值，每次加上一个0 到 20 之间的随机数random ，直到 B 的值大于或等于 100 循环才结束 . The number of iterations actually performed in the loop will vary depending on the random numbers. The check is performed at the start of the loop - so if the condition is already true, zero iterations may be performed. Take the ; away from the second line to see this happen. 
  Define.i B
  ; B = 100
  While B < 100
�   B + Random(20)
�   Debug B
  Wend
This loop is very similar to the last except that the check is performed at the end of the loop. So one iteration, at least, will be performed. Again remove the ; from the second line to demonstrate.

  ; C = 100
