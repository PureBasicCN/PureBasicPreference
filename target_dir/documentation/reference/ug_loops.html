用户指南 - 循环

数据、 事件或许多其他的事情，可以使用循环来处理，不断检查是否满足执行条件。循环控制使用的关键词: Repeat : Until, Repeat : Forever, While : Wend, For : Next, ForEach : Next.

在此计数器中 每次循环 A 将增加 2 ，这个循环将总是执行相同数量的迭代次数。 
  Define.i A
  For A = 0 To 10 Step 2
  ?Debug A
  Next A
这个循环，将变量 B 的值，每次加上用random产生的一个0 到 20 之间的随机数，直到 B 的值大于或等于 100 循环才结束。实际执行循环中的迭代次数根据随机数字会有所不同。在循环开始时会执行检查，当随机数为 0 时 ，If 条件为True，条件成立， 循环将结束. 加上 ; 分号，从第二行开始演示。 
  Define.i B
  ; B = 100
  While B < 100
?B + Random(20)
Debug B
  Wend
这种循环和上面的非常相似，只是在循环体结束时执行检查。所以至少，将执行一次循环。再次删除 ; 分号，从第二行演示. 
  Define.i C
  ; C = 100
  Repeat
C + Random(20)
Debug C
  Until C > 99
这种循环是无限循环。它不会停止，直到您停止它 （IDE 工具栏上使用红色的 X 按钮）。 
  Define.i D
  Repeat
Debug D
  ForEver
还有一个特殊的循环，用于lists与maps，反过来，它将循环访问每个成员的列表list （或地图map）。 
  NewList Fruit.s()
  
  AddElement(Fruit())
  Fruit() = "Banana"
  
  AddElement(Fruit())
  Fruit() = "Apple"
  
  AddElement(Fruit())
  Fruit() = "Pear"
  
  AddElement(Fruit())
  Fruit() = "Orange"
  
  ForEach Fruit()
Debug Fruit()
  Next Fruit()
