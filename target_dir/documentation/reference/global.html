Global


语法
Global[.<type>] <variable[.<type>]> [= <expression>] [, ...]
概要
Global 提供将变量定义为全局的能力,定义的变量随后可以在 Procedure内部使用. 在这种情况下 Global 命令必须在定义过程 之前 将相关变量定义为全局。每个变量可能具有直接分配给它的默认值。如果在Global之后为变量指定了类型，通过这一声明更改默认类型。Global 也可以用于 数组, 列表 和 映射.

针对过程内部使用的局部变量, 有和全局变量名称相同的，请查看Protected和Static关键字。

示例︰变量

  Global a.l, b.b, c, d = 20
  
  Procedure Change()
    Debug a  ; Will be 10 as the variable is global
  EndProcedure
  
  a = 10
  Change()

Example: With array

  Global Dim Array(2)
  
  Procedure Change()
    Debug Array(0)  ; Will be 10 as the array is global
  EndProcedure
  
  Array(0) = 10
  Change()

示例︰默认类型

  'Angle' 和 'Position' 将是一个浮点数，因为他们没有指定类型
  ;
  Global.f Angle, Length.b, Position
