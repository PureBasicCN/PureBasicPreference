For : Next


语法
For <variable> = <expression1> To <expression2> [Step <constant>]
  ...
Next [<variable>]
概要
For : Next is used to create a loop within a program with the given parameters. At each loop the <variable> value is increased by a 1, (or of the "Step value" if a Step value is specified) and when the <variable> value is above the <expression2> value, the loop stop.

With the Break command its possible to exit the For : Next loop at any moment, with the Continue command the end of the current iteration can be skipped.

For : Next 循环制适用于整数值， 在 Step 常数的表达式中. Step 常量也可以是负数.

示例

  For k = 0 To 10 
    Debug k
  Next
在此示例中，程序将循环 11次 (0 到 10)，然后退出。

示例

  For k = 10 To 1 Step -1
    Debug k
  Next
在此示例中，该程序将循环 10 倍次(10 到 1 向后)，然后退出。

示例

  a = 2
  b = 3 
  For k = a+2 To b+7 Step 2
    Debug k
  Next k
Here, the program will loop 4 times before quitting, (k is increased by a value of 2 at each loop, so the k value is: 4-6-8-10). The "k" after the "Next" indicates that "Next" is ending the "For k" loop. 如果使用另一个变量，则编译器将生成一个错误。它可以用于嵌套"For/Next" 循环.

示例

  For x=0 To 10 
    For y=0 To 5
      Debug "x: " + x + " y: " + y
    Next y
  Next x
注意: Be aware, that in PureBasic the value of <expression2> ('To' value) can also be changed inside the For : Next loop. 用错时这可能导致无限循环。
