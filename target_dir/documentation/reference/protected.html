Protected


语法
Protected[.<type>] <variable[.<type>]> [= <expression>] [, ...]
概要
Protected 允许一个 变量 的访问仅限 Procedure 内，如果是同样的变量定义为 Global 在主程序中时Protected的功能通常在其他BASIC中被称之为"Local"(本地变量). 每个变量可以直接分配一个默认值给它. 如果类型在Protected之后指定,则默认类型被换成此定义. Protected 同样可以用于 数组, 列表 和 映射.

本地变量的值将在每个过程调用重新初始化。要避免此问题，可以使用关键字Static从局部变量分离全局变量并且保持他们的值不变。

示例︰用于变量

  Global a
  a = 10
  
  Procedure Change()
    Protected a
    a = 20
  EndProcedure
  
  Change()
  Debug a ; 将打印10,因为变量已经被保护.

示例︰与数组

  Global Dim Array(2)
  Array(0) = 10
  
  Procedure Change()
    Protected Dim Array(2) ; 此数组被保护，它将是本地变量.
    Array(0) = 20
  EndProcedure
  
  Change()
  Debug Array(0) ; 将打印 10,因为数组已经被保护.
