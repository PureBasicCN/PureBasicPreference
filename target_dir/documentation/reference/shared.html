Shared


语法
Shared <variable> [, ...]
概要
Shared 允许 变量, 数组, 列表或者 映射在 过程内访问. 当 Shared使用数组、列表或者映射时，只能在名称后指定 "()' .

示例︰用于变量

  a = 10
  
  Procedure Change()
    Shared a
    a = 20
  EndProcedure
  
  Change()
  Debug a   ; 将打印 20, 因为变量被共享.

示例︰用与数组和列表

  Dim Array(2)
  NewList List()
  AddElement(List())
  
  Procedure Change()
    Shared Array(), List()
    Array(0) = 1
    List() = 2
  EndProcedure
  
  Change()
  Debug Array(0)  ;将打印 1, 因为数组被保护.
  Debug List()    ; 将打印2, 因为列表被保护了.
