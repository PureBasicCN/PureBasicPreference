IsNode()

语法

Result = IsNode(#Node)
概要
Tests if the given node is valid and correctly initialized.

参数

#Node The node to test.

返回值

Nonzero if the node is valid, zero otherwise.

备注

此函数是保险性的，可能与任何值一起使用。This is the correct way to ensure a node is ready to use.
