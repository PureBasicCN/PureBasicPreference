IsFont()

语法

Result = IsFont(#Font)
概要
Tests if the given #Font is a valid and correctly initialized font.

参数

#Font The font to use.

返回值

Returns nonzero if #Font is a valid font and zero otherwise.

备注

此函数是保险的并且可以与任何值一起使用。If Result is not zero then the object is valid and initialized, otherwise it returns zero. This is a good way to check that a font is ready to use.
