GetDatabaseLong()

语法

Result = GetDatabaseLong(#Database, Column)
概要
Returns the content of the specified #Database column as an integer number. 此命令只有在FirstDatabaseRow()、 PreviousDatabaseRow()或NextDatabaseRow()等函数调用成功后才有效。

参数

#Database 要使用的数据库。
Column 要使用的列。DatabaseColumnIndex() is available to get the index of a named column.

返回值

Returns the content of the column as an integer value.

备注

若要确定列的类型，可以使用DatabaseColumnType() 。

注意︰ 每列此函数可以是只调用一次。因此如果需要超过一次使用此值，数据必须存储在变量中，因为所有的后续调用将返回错误值。This is an ODBC limitation.

参阅

GetDatabaseBlob(), GetDatabaseDouble(), GetDatabaseFloat(), GetDatabaseString(), GetDatabaseQuad()
