PureBasic - Database

概要

数据库函数库是一个简单设置函数来访问 SQLite, PostgreSQL 或者任何数据库类型(Oracle, MySQL, Access, 等等) via ODBC. 访问和更新数据使用 SQL 查询，因此它完成的是必须了解 SQL 语法。

以下是一些有关 SQL 语法的链接︰
W3Schools SQL Tutorial
SQLite SQL 函数
PostgreSQL 手册

数据库编程开始前需要由UseODBCDatabase()、 UseSQLiteDatabase()和UsePostgreSQLDatabase()来初始化的数据库环境。

注意: 在 Windows 中使用 ODBC 数据库，, 就必须建立 ODBC"用户数据源"，因此你的数据库源 ODBC必须有效 并且此数据库必须可用。更多的信息，请参阅 Windows 的 ODBC 帮助文档。
