StrU()

语法

Result$ = StrU(Value.q, Type)
概要
Converts an unsigned numeric number into a string.

参数

Value.q The value to convert.
Type 它可以是以下值之一︰ 
  #PB_Quad ︰ 值作为Quad来处理，范围从 0 到 18446744073709551615 （默认值）
  #PB_Byte ︰ 值作为Byte来处理，介于 0 到 255
  #PB_Ascii ︰ 值作为 ascii 字符来处理，范围从 0 到 255 
  #PB_Word ︰ 值作为word处理，范围从 0 到 65535 之间
  #PB_Unicode︰ 值作为一个 unicode 字符处理，从 0 到 65535 之间
  #PB_Long ︰ 值作为一个long 来处理，范围从 0 到 4294967296

返回值

A string holding the converted value.

备注

Signed integer numbers must be converted with Str() and float numbers with StrF() or StrD().

示例

  byte.b = 255
  Debug Str(byte) ; Will display -1
  Debug StrU(byte, #PB_Byte) ; Will display 255

参阅

Str(), StrD(), StrF(), Val(), ValD(), ValF(), Hex(), Bin()
