Hex()

语法

Result$ = Hex(Value.q [, Type])
概要
以十六进制格式将quad转换为字符串。

参数

Value.q 要转换的数字。
Type (可选) 如果值应处理作为另一种类型，可以指定以下值之一︰ 
  #PB_Quad ︰ 值作为Quad来处理，范围从 0 到 18446744073709551615 （默认值）
  #PB_Byte ︰ 值作为Byte来处理，介于 0 到 255
  #PB_Ascii ︰ 值作为 ascii 字符来处理，范围从 0 到 255 
  #PB_Word ︰ 值作为word处理，范围从 0 到 65535 之间
  #PB_Unicode︰ 值作为一个 unicode 字符处理，从 0 到 65535 之间
  #PB_Long ︰ 值作为一个long 来处理，范围从 0 到 4294967296

返回值

保存指定的值的十六进制表示形式字符串。

示例

  Debug Hex(12) ; Will display "C"
  Debug Hex(1234567890) ; Will display "499602D2"

备注

如果需要在输出字符串中前导零，使用RSet()函数，如下所示︰ 
  Debug RSet(Hex(12), 4, "0") ; 将显示 "000C"

示例

  Debug Hex(-1)
  Debug Hex(-1, #PB_Byte)
  Debug Hex(-1, #PB_Word)
  Debug Hex(-1, #PB_Long)
  Debug Hex(-1, #PB_Quad)    ; quad 值是默认的
