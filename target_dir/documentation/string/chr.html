Chr()

语法

Text$ = Chr(CharacterValue)
概要
返回一个用指定字符创建的字符串。

参数

CharacterValue 字符的值。如果作为Unicode 模式编译该程序，它可以是 ASCII 值或 unicode 值。

返回值

返回一个用指定字符创建的字符串。

示例

  Debug Chr(33) ; 将显示 "!"

备注

所有的 Ascii 值与他们有关的数据的表可以在这里查看。

此命令还可以在Unicode 模式下工作，然后返回与指定值相关联的字符。 
  Debug Chr(401) ; 将显示 "?"
  Debug Chr(956) ; will display "?quot;
