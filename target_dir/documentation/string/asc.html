Asc()

语法

Result = Asc(String$)
概要
返回指定字符串的第一个字符值。

参数

String$ 要获取的第一个字符值的字符串。

返回值

字符串第一个字符值。如果该程序被编译成unicode，它将返回 unicode 字符的值，否则它将返回一个 ASCII 字符值。

示例

  Debug Asc("!") ; will print '33'

备注

同样，也可用放在单引号之间获得一个常量字符值。

示例

  Debug '!' ; will print '33'
