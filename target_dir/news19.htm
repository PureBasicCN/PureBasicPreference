PureBasic V2.90 (AmigaOS) :


- Compiler core updated to 2.90
- Procedure address support (a = @Procedure())
- !AsmKeyword for raw access to PhxAss
- Optimized code output
- Logical And/Or changed (faster)
- Faster compiling
- Variables, Constant, Arrays are now case insensitive (easier) ('myvariable' is the same as 'MYVARIABLE' or 'MyVarIable')
